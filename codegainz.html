Codegainz Blog

View Site View Code

Codegainz is a Gatsby-powered React site that functions as my official blog. While in its beginning stages, this is a long-term project that I am excited about maintaining. The long-term goal for this site is to transform it into a learning platform for aspiring developers.

This project is made with Gatsby (React static generator), Contentful CMS (content management system), and is hosted with Netlify. The site is fully responsive for all sizes of screen, from ultrawide monitors to extra small phones. The site has lazy loading for images, code-splitting, and is optimized for markdown. In the current iteration of this site, I've implemented Amazon affliate linking as a test before adopting it on a client's site. Click on either the image or "View Site" button above to see the live site, or click the "View Code" button above to view the code on GitHub.
