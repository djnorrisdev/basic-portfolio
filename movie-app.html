React/Redux Movie App

View Site View Code

This movie site uses The Movie Database (TMDB) API to allow guests and authenticated users to browse movies and tv shows by typing keywords in the search box on the home page. Users have an option to login using Google Authentication. When viewingthe results cards on the home page, authenticated users have the option of adding that favorite to their user space in the Firebase database by clicking the star icon in the card.

Authenticated users can also view and modify their favorites list through the user dashboard page. User's favorites are stored in the dashboard indefinitely, allowing the user to log back in after an inactivity period and still have access to their favorites.

This project is made with React, Redux, Firebase, Google Auth, Styled Components, TMDB API, and is hosted with Zeit Now. The site is fully responsive for all sizes of screen, from ultrawide monitors to extra small phones. Click on either the image or "View Site" button above to see the live site, or click the "View Code" button above to view the code on GitHub.
