Music Tribute Site

View Project View Code

Browse this music tribute site to see biography information, new album information, and current tour dates. Click on the embeded YouTube videos to watch Diplo's best videos without leaving the current page. The page is fully responsive for all sizes of screen, from ultrawide monitors to extra small phones.

This static site was created using HTML, SaSS, and jQuery. Gulp was used for compiling Sass and minification of CSS. Click on either the image or "View Project" button above to see the live project, or click the "View Code" button above to view the code on GitHub.
